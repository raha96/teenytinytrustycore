# TeenyTinyTrustyCore (3TC)

TeenyTinyTrustyCore is an hardware root of trust (HRoT) designed solely using CogniChip. The two folders, [approach1](approach1/FINAL_STATUS.md) and [approach2](approach2), contain the results of our two attempts at realizing this HRoT.
